Análise de Código

SonarQube

É uma plataforma de análise estática de código fonte que fornece insights detalhados sobre a qualidade do código em um projeto de software. Ele realiza uma variedade de verificações automatizadas em relação a boas práticas de programação, padrões de codificação, vulnerabilidades de segurança e bugs potenciais.

ESLint

O ESLint é uma ferramenta de análise estática de código para JavaScript, usada principalmente para identificar e relatar padrões problemáticos ou suspeitos encontrados no código JavaScript. Ele é altamente configurável e permite que você defina regras personalizadas para atender às necessidades específicas do seu projeto.

Bitbucket

Bitbucket é uma plataforma de hospedagem de código fonte baseada em Git, fornecida pela Atlassian. Ela oferece controle de versão para equipes de desenvolvimento, permitindo que os usuários armazenem, colaborem e gerenciem o código fonte de seus projetos de software.

Snyk

Snyk é uma plataforma de segurança de código que ajuda as equipes de desenvolvimento a identificar, corrigir e prevenir vulnerabilidades de segurança em seus aplicativos e dependências de código aberto.

FindBugs

Uma ferramenta de análise estática de código para Java que identifica bugs potenciais, como vazamentos de memória, acesso indevido a variáveis e uso incorreto de APIs.

Planejamento e Modelagem

Draw.io

O Draw.io é uma ferramenta online intuitiva para criação de uma variedade de diagramas, como fluxo de processos e UML. Com uma interface de arrastar e soltar, é fácil de usar, permitindo colaboração em tempo real e acesso remoto. Ideal para planejamento e modelagem de projetos.

Microsoft Visio

O Microsoft Visio é uma ferramenta de diagramação amplamente utilizada para criar uma variedade de diagramas, como fluxogramas, organogramas, mapas mentais e diagramas de rede. Ele oferece uma ampla gama de formas, modelos e estilos de formatação para atender às necessidades de diferentes projetos e equipes.

Creately

É uma plataforma de diagramação online que oferece uma ampla variedade de recursos para criar diagramas de forma rápida e fácil. Além disso, o Creately é baseado na nuvem, o que significa que os usuários podem acessar e editar seus diagramas de qualquer lugar, a qualquer momento.

Visual Paradigm

Uma suíte de ferramentas de modelagem que oferece suporte a uma variedade de notações, incluindo UML, BPMN, SysML e muito mais. Ele oferece recursos para modelagem de software, modelagem de negócios e engenharia de sistemas.

Ardoq

Uma plataforma de modelagem de arquitetura corporativa que permite criar modelos visuais de arquitetura de negócios, sistemas de TI e processos de negócios.

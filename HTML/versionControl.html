Controle de Versão

git

O Git é um sistema de controle de versão distribuído, ou seja, ele permite que você mantenha um histórico completo de todas as alterações feitas no código-fonte do seu projeto, além de permitir que você trabalhe de forma colaborativa com outros desenvolvedores.

GitLab

O GitLab é uma plataforma abrangente de DevOps, integrando controle de versão,integração contínua, entrega contínua, gerenciamento de projetos e rastreamento de problemas. Ele simplifica o desenvolvimento de software, permitindo colaboração eficiente, automação de processos e entrega rápida de software de alta qualidade.

AWS CodeCommit

O AWS CodeCommit é um serviço de hospedagem de repositórios de controle de versão baseado em Git, fornecido pela Amazon Web Services (AWS). Ele oferece um ambiente seguro e escalável para armazenar e gerenciar o código fonte de aplicativos de software na nuvem da AWS.
